Die roll percentage calculator

This web app will calculate the chance of rolling equal to or above every possible result, when rolling a given number of dice with a given number of sides per die.

This app was created using vanilla HTML5, CSS3, and JavaScript.
Basic styling for the page was provided by Jeff George's own open-source stylesheet, drop-in.css.
No other frameworks or libraries were used to create this app.
