About my work

Jeff George's Portfolio

The work you'll find linked to this page includes a variety of projects I've developed, either independently or as assignments during my time at Dev Bootcamp, the original immersive web-development training program. These projects demonstrate my proficiency in a wide variety of web development technologies, including HTML5, CSS3, JavaScript, Ruby, Rails, and SQL.

When presenting web pages and applications that can be displayed within this website, I've included a sidebar describing the parameters and purpose of the project, and the technologies used to produce it. When linking to pages or entire sites hosted elsewhere on the web, I've provided a page on this site that describes the project and technologies, along with a link directly to the site.

The source code for all these projects is publicly available at my repository on GitHub. For most projects, you can review the live source code by using the Developer's Tools in your browser. So you can more easily see the quality my work, the source code for these projects has not been minified.
