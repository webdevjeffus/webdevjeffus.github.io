About my website

As of Oct. 19, 2015

This website is written in vanilla HTML5 and CSS3, on a Mac Book Pro, using the Sublime 2 text editor. Images were prepared using GIMP, the Gnu Image Manipulation Program. Mock-ups for this homepage and several interior pages were prepared using Balsamiq 3 wireframing package. Fonts include Permanent Marker (banner headings), Walter Turncoat (sub-headings), and Open Sans (body text), which are all served by Google Fonts. Icons for social media and navigation are displayed using Font Awesome, which delivers icons as scalable, stylable fonts.

The framework used to create this website is griddy, my own ultra-lightweight, grid-based responsive design template, which is available for use under the GNU General Public License. Griddy relies upon the normalize.css for its resets, and borrows certain elements from the HTML5 Boilerplate front-end template.
